# Changelog

## v0.1.0-alpha - 2026-06-28

Initial alpha release.

### Added

- Next.js App Router web app.
- Supabase offers migration and seed data.
- `GET /api/offer` endpoint for one active offer.
- Admin Offers page for creating, editing, disabling, and deleting offers.
- Node.js CLI package.
- Claude Code status line support.
- Demo landing page with GIF and screenshots.
- Project architecture docs in `CLAUDE.md`.
- MIT license and contribution guide.
