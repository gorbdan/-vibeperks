# MVP-документ проекта

VibePerks показывает короткие полезные AI-предложения в статусной строке во время работы Claude Code / Codex / Cursor.

MVP включает:

- Next.js web app;
- Supabase Postgres schema;
- admin page для управления offers;
- public API endpoint для получения активного offer;
- endpoint для записи impression;
- endpoint для записи click;
- Node.js CLI script для Claude Code status line.

Главный принцип: максимально простой код, без лишних функций.
