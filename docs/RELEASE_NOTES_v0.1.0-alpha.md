# v0.1.0-alpha Release Notes

VibePerks `v0.1.0-alpha` is the first public-ready alpha.

## Highlights

- Demo landing page that explains the product quickly.
- Offer storage through Supabase.
- Public API endpoint for one active offer.
- Admin Offers management page.
- CLI output for terminal and Claude Code status line usage.

## Not Included Yet

- Admin authentication.
- Impression tracking.
- Click tracking.
- Published npm package.
- Hosted production deployment.
